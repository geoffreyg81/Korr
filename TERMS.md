## 3. Description du service

Korr analyse un texte fourni par l'utilisateur et propose une version corrigée.

**Le traitement est intégralement local.** Le texte est analysé dans le
navigateur ou sur l'ordinateur de l'utilisateur. Aucun texte n'est transmis à un
serveur de l'éditeur : il n'en existe aucun.

Le mode de correction approfondie repose sur un modèle de langue exécuté
localement, que l'utilisateur installe lui-même. Ce mode est facultatif et, lui
aussi, sans transmission externe.
